Maven Cactus plugin

This is a plugin for Maven that allows to easily perform Cactus unit-testing with Maven. See the features list for more details.

News

Date Description
28 Janv 2004 Maven plugin 1.7 for Cactus is available. See the changes report for the detailed list of changes. To install it, run:
maven plugin:download -DartifactId=cactus-maven -DgroupId=cactus -Dversion=1.7
Read the using guide for more details on installation and usage.
